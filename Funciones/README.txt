Funcion variable_de_retorno <- Nombre ( Argumentos )
	{cuerpo_funcion}
Fin Funcion


- {variable_de_retorno}: si su función/subproceso calcula y devuelve algún valor, complete aquí el nombre de la variable que utilizará para almacenar 
ese valor; si su función/subproceso no devuelve nada puede eliminar la variable y la flecha de asignación. 

- {nombre}: introduzca aquí el nombre que recibirá la función/subproceso. 

- {argumentos}: introduzca aquí la lista de argumentos que requiere su función, utlizando comas para separarlos, e ingresando el identificador para 
cada uno y opcionalmente las palabras clave "por valor" o "por referencia" para indicar el tipo de pasaje; si la función/subproceso no requiere 
argumentos puede dejar este campo en blanco, y opcionalmente omitir los paréntesis.
