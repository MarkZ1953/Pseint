Crear un programa en PseInt que despliegue un menú de opciones que permite al usuario realizar diversas conversiones de números. En el menú, se ofrecen 
cinco opciones: 

1 ) convertir un número decimal a binario
2 ) convertir un número decimal a hexadecimal
3 ) convertir un número decimal a octal
4 ) verificar si un número es decimal o no  
5 ) salir del programa.

El usuario puede elegir la opción que desee ingresando el número correspondiente. Si elige la opción de convertir un número decimal a binario, 
se le solicita ingresar el número que desea convertir, y el programa realiza la conversión y muestra el resultado. Si elige la opción de convertir un 
número decimal a hexadecimal u octal, se le solicita ingresar el número, y el programa realiza la conversión y muestra el resultado.

Si el usuario elige la opción de verificar si un número es decimal o no, se le solicita ingresar el número, y el programa verifica si el 
número contiene un punto decimal o una coma. Si lo contiene, se muestra un mensaje indicando que el número es decimal, de lo contrario, se muestra un 
mensaje indicando que el número no es decimal.
